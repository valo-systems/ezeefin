## Headline findings
1. **ezeefin.co.za is down (HTTP 403)** — the domain resolves but serves nothing; recovering registrar/hosting access is the first blocking task.
2. The old site was a 2015 Joomla template whose archived copy shows **injected spam text** — rebuild, don't rescue.

6. Recommended concept: a **vehicle-buying concierge with a customer progress tracker** — no observed SA competitor offers one.
